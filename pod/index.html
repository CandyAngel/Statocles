NAME

Statocles::Help - Get help for Statocles

GUIDES

CONFIGURATION

To get started with your own Statocle site, run the statocles create command, or see the Statocles config help in Statocles::Help::Config.

CONTENT

To use Statocles to manage content, see Statocles::Help::Content.

THEMING

To change how your Statocles site looks, see Statocles::Help::Theme.

DEPLOYING

To deploy your Statocles site to a Git repository, or any remote server, see Statocles::Help::Deploy.

DEVELOPING

To develop custom Statocles applications, custom ways to deploy, custom template languages, or other extensions, see Statocles::Help::Develop.
