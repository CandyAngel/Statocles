Release v0.033

Tags: release

Posted on 2015-01-11 by preaction

Just some minor bugfixes in this release: The --date option when making a new blog post finally works, and when writing a new post while the daemon is running, the site will be updated as expected.

Full changelog below.

  • fix unable to set date for blog post from command (#198)
  • build a new temp site for each command test
  • remove root dotfiles and root ini files from dist
  • switch to dzil Git::GatherDir
  • add new blog posts to store so auto-build works (#183, #180)
  • "now" is easily mistaken for "not" in test diag
